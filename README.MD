### Запуск проекта 
 
Установка зависимостей 
 
```bash 
$ pip install -r ./requirements.txt 
$ python ./main.py 
```

### Инофрмация о боте 
 
У бота существует 3 команды: /start, /help, /search 
 
/start - запуск бота и приветствие 
/help - инструкция по использованию бота 
/search (args) - поиск по вашу запросу
